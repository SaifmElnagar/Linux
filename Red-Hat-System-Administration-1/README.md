## 📚 Course Chapters

This repository follows the **15 chapters** of the course:

### 1️⃣ Intro  
Introduction to the course, Linux basics, and the overall learning path.

### 2️⃣ Accessing the Command Line  
Working with the Linux shell, terminal basics, and essential command-line interaction.

### 3️⃣ Managing Files from the Command Line  
Creating, moving, copying, deleting, and organizing files and directories using CLI tools.

### 4️⃣ Getting Help in Red Hat Enterprise Linux  
Using built-in help resources such as `man`, `help`, and documentation tools.

### 5️⃣ Creating, Viewing, and Editing Text Files  
Working with text files and editors, and learning how to read and modify file content efficiently.

### 6️⃣ Managing Local Users and Groups  
Creating and managing users and groups, and understanding account administration in Linux.

### 7️⃣ Controlling Access to Files  
Understanding file permissions, ownership, and access control mechanisms.

### 8️⃣ Monitoring and Managing Linux Processes  
Viewing, monitoring, and controlling processes running in the system.

### 9️⃣ Controlling Services and Daemons  
Managing background services and understanding how daemons operate in Linux.

### 🔟 Configuring and Securing SSH  
Using SSH for secure remote access and applying basic SSH security practices.

### 1️⃣1️⃣ Analyzing and Storing Logs  
Understanding system logs, log analysis, and where important system information is stored.

### 1️⃣2️⃣ Managing Networking  
Configuring and troubleshooting network settings in Linux systems.

### 1️⃣3️⃣ Archiving and Transferring Files  
Using Linux tools to archive, compress, and transfer files between systems.

### 1️⃣4️⃣ Installing and Updating Software Packages  
Managing software installation, updates, and package repositories.

### 1️⃣5️⃣ Accessing Linux File Systems  
Understanding Linux storage structure and working with file systems and mount points.
